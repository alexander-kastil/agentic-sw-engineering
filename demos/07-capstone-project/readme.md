# Capstone Project: End-to-End Agentic Development

## From Planning to Production

This capstone project demonstrates the complete agentic software engineering lifecycle—from initial planning and proof of concept through implementation, modernization, testing, and documentation. You will build a real application using AI-assisted development, making strategic decisions about architecture, technology choices, and deployment strategies.

| Phase                                                      | Description                                                                                                      |
| ---------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| **[Planning & Proof of Concept](./01-planning/)**          | Define requirements, architecture, and proof of concept. Validate technology choices before committing to build. |
| **[Orchestrating & Implementation](./02-implementation/)** | Build the application using Copilot-assisted development with focus on core features and functionality.          |
| **[Upgrading & Modernization](./03-upgrading/)**           | Modernize the application across framework versions and demonstrate migration paths to current technologies.     |
| **[Testing using Copilot](./04-testing/)**                 | Validate functionality with unit tests, integration tests, and end-to-end scenarios.                             |
| **[Using Copilot for Documentation](./05-docs/)**          | Document architecture, APIs, deployment guides, and operational runbooks for production readiness.               |
