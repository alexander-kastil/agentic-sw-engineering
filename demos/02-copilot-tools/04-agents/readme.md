# Agents

Agents are specialized AI personas configured with specific tools, instructions, and handoffs to handle targeted development tasks. Learn about agent concepts, explore repository agents, and set up Claude agents.

## Sections

[01-agents-overview](01-agents-overview/) covers agent fundamentals, why to use them, and how handoffs enable guided multi-step workflows. Start here to understand agent concepts and sample configurations.

[02-repo-agents](02-repo-agents/) documents the agents configured in this repository, each optimized for specific development tasks like deployment, testing, or pipeline management.

[03-claude-agents](03-claude-agents/) explains how to enable and use Claude agents powered by Anthropic's Claude for autonomous agentic workflows with their own built-in tool set.
