# Update Agent Framework to Latest Version

This demo shows how to upgrade Python agent implementations to the latest Microsoft Agent Framework libraries and Azure AI Foundry packages. You will examine existing agent files, identify outdated dependencies, and update project configurations to leverage the newest features and improvements.

The workflow includes checking Microsoft Learn MCP for library updates, modifying dependency files, creating a Python environment, and generating a detailed upgrade report documenting all changes made.

## Prompt

```
Examine demos\03-agentic-coding\01-local\03-update\agentfw_tools-knowledge-py. Use your microsoft learn mcp to check if there are updated libraries available that allow updating all agents implemented in files with prefix agentfw_ this to the latest libraries and show the agents in the new Microsoft Foundry. Your tasks:
Pin down the library version to latest possible
Update pyproject.toml and requirements.txt. No need to upgrade pip
Create and activate the Python environment and install dependecies
Create an upgrade-report.md in the folder
```
