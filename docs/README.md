# Documentation

Welcome to the comprehensive documentation for **Agentic Software Engineering using GitHub Copilot** — a master class by [Alexander Kastil](https://www.integrations.at/kontakt-impressum.aspx).

## Overview

This course is a 3-day, hands-on journey covering the full spectrum of agentic software engineering. It equips software engineers, architects, and technical leaders with the knowledge and practical experience to leverage GitHub Copilot and AI agents in modern software development.

- **Duration**: 3 days / 24 hours
- **Split**: 70% instruction & demos · 30% labs
- **Audience**: Software Engineers, Architects, Team Leads

## Course Modules

| Module | Topic | Demos |
|--------|-------|-------|
| [Module 1](../demos/01-fundamentals/) | GitHub Copilot Fundamentals | Codespaces, inline suggestions, slash commands, context variables, PRs |
| [Module 2](../demos/02-copilot-tools/) | GitHub Copilot Artifacts & Tools | Instructions, prompt files, MCP, custom agents, skills, memory, hooks |
| [Module 3](../demos/03-agentic-coding/) | Implementing Agentic Coding | Local agents, cloud agents, background agents, orchestration, Claude Code |
| [Module 4](../demos/04-advanced-topics/) | CLI, SDK & Advanced Topics | Copilot CLI, agentic workflows, Copilot SDK, MCP apps |
| [Module 5](../demos/05-agentic-devops/) | Agentic DevOps | Azure CLI, IaC (Bicep & Terraform), Azure DevOps Pipelines, GitHub Actions |
| [Module 6](../demos/06-spec-driven-dev/) | Spec-Driven Development | Spec-driven workflow, constitution, technical plan, GitHub Spec Kit |
| [Module 7](../demos/07-capstone-project/) | Capstone Project | End-to-end agentic development: planning, implementation, testing, docs |

## Getting Started

1. **Prerequisites**: Professional software development experience
2. **Environment**: Set up [GitHub Codespaces](../demos/01-fundamentals/00-codespaces/) or a local dev container
3. **Start learning**: Begin with [Module 1 — GitHub Copilot Fundamentals](../demos/01-fundamentals/)
